# 📋 Консольный менеджер задач (Task Tracker)

Мой первый проект на Java — простая программа для создания и просмотра списка задач прямо из консоли.

## 🚀 Возможности

- Создание новой задачи с названием и описанием
- Автоматическая генерация ID задачи (случайное число)
- Автоматическая установка текущей даты создания
- Сохранение всех задач в текстовый файл `Tasks.txt`
- Просмотр всех сохранённых задач

## 🛠️ Технологии

- Java 8+
- Работа с файлами (java.io)
- Стандартная библиотека Java (без внешних зависимостей)

## ▶️ Как запустить

1. Убедитесь, что у вас установлен JDK (версия 8 или выше)
2. Скачайте или склонируйте репозиторий
3. Откройте терминал в папке проекта
4. Скомпилируйте:
   ```bash
   javac src/*.java
