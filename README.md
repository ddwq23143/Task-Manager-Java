# Task-Manager-Java
Простое приложение для создания и просмотра задач прямо в консоли. Пользователь вводит название и описание задачи, программа автоматически присваивает ей уникальный ID, ставит текущую дату и сохраняет всё в текстовый файл Tasks.txt. Также можно посмотреть список всех созданных задач.
